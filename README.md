# Inventory Management App - RESTful API

A lightweight, robust RESTful API built with Node.js and Express to manage inventory items using complete CRUD capabilities.

## 🛠️ Features
* **Create:** Add new items to the inventory track (`POST`).
* **Read:** Retrieve individual items or the entire inventory array (`GET`).
* **Update:** Alter details, pricing, and stock levels seamlessly (`PUT`).
* **Delete:** Remove discontinued goods instantly from the records (`DELETE`).
